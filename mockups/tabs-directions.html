Vertex · the other four tabs

Two languages, your call per tab

Same Cold Archive palette and type system as the Core tab, now extended to the rest of the sheet, using Mara Quill's real data. For each tab you get Option A — The Ledger (editorial, borderless, hairline-divided, big serif) and Option B — The Index (structured two-column / card entries, denser, scannable). You can pick a different option per tab. Veto and remix freely.
